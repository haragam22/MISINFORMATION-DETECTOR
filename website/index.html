Why this matters

Misinformation spreads faster than reliable information. TruthLens demonstrates a simple, transparent workflow — retrieve, score, and explain — so users not only receive a verdict, they also learn why.

Outputs you'll always see
  • Verdict & confidence
  • Top evidence snippets (clickable)
  • Signal breakdown (how score was computed)
  • Short education tip specific to the claim

How it works

1. Claim extraction

User highlights text or pastes a claim. The system extracts the factual statement.

2. Evidence retrieval

Backend searches fact-check databases, reputable media, and scientific sources (or returns canned results for demo reliability).

3. Scoring & explanation

Weighted signals (source reliability, corroboration, stance, temporal checks) are combined into a score; the system shows the breakdown and an education tip.

Want to try it in your browser?

Use the Chrome extension for quick highlight-and-check. The repo contains the extension and backend code.
